Detailed Description

Defines TextBox and TextBox which are used to display dynamic text within a canvas.

This file contains a number of of type definitions that are used by TextBox and TextBox constructors. The intent here is to greatly simplify the use of TextBox by providing users with a collection of pre-set configurations to choose from.
