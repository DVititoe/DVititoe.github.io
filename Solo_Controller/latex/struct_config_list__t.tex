\doxysubsection{Detailed Description}
Data structure defining all \doxylink{class_text_box}{Text\+Box} used by all menus. 

The static method \doxylink{class_dynamic_text_aebf272bd8cd47b9870574ecfc4004720}{Dynamic\+Text\+::initialize\+Fonts()} is used to initialize this data structure. This method must be called before intantiating any \doxylink{class_dynamic_text}{Dynamic\+Text} objects.
