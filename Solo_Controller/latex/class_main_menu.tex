\doxysubsection{Detailed Description}
Main menu for the Solo\+Controller application. 

This menu provides the primary user interface for the Solo\+Controller application. Note that the callback functions used by \doxylink{class_main_menu}{Main\+Menu} are contained within the \href{namespace_c_b_1_1_m_a_i_n.html}{\texttt{ CB\+:\+::\+MAIN namespace}}. This object uses significant DRAM resources. So, it has been implemented as a Singleton object.
