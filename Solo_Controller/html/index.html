Touch Screen Homebrew BIAB Controller

This repository describes the design and specifications for a new touch screen controller, designed to operate an 240V electric Brew in a Bag (BIAB) system. I specifically designed this controller for use with a cutstomized Spike Solo Brew System, but this this controller can be used with any compatable 240v Electric Brew Kettle.

*

Concept for the controller's User Interface. This consists of a touch screen to manage the brew system, two physical switches (Element Enable and Power Switch), and an Alarm.

*

Why design a new controller? After all, there are a number of BIAB controllers already out there, ready to to purchased and used right out of the box. None of these controllers had all of the features that I wanted for my brew system. So, I decided to build my own.

Key Design Features

Some of the key design features that differentiate this controller from others include:

  • Two Temperature Probes: The controller has built in support for two RTD temperature probes. One probe measures Kettle temperature. The other temperature probe can be installed on the output of your wort chiller.
  • Kettle Volume Measurement: The controller is designed to work with a tri-clamp mounted pressure sensor installed onto the kettle. The controller uses the pressure values measured by tthis sensor to estimate the amount of wort in the kettle.
  • Stuck Mash Alarm: This design takes advantage of a unique feature of the Spike Solo system. Should a stuck mash occur, the wort volume level in the main kettle will fall, as more and more wort is pumped into the stainless steal Spike sparge basket. When the wort level in the main kettle falles below a user defined value, an alarm sounds.
  • Proportaional Temperature Control in the Kettle: Most controllers use a standard SSR relay to control the heating element within the kettle. These relays alternate between fully turning on (or off) the heating element. This design uses a proportional SSR relay to control the kettle heating element. Proportional relays allow the controller to apply just enough power to maintain kettle temperature. In my particular case, I was interested in experimenting with longer temperature controlled hop stands. The proportional relay makes it possible to control kettle temperature while minimizing the risk of hops isomerization.
  • Physical Element Disable Switch: Most touch screen controllers rely soley upon signals from it's microcontroller to enable or disable the heating element. I prefer a physical switch, that will electrically disable the element when needed. This guarantees that a microcontroller or software glitch won't prevent you from disabling the element.
  • Loud Ass Alarm: Most conrollers have fairly weak alarms. This design uses a 12v DC alarm from Auber Instruments to generate 85 decibels. There is no chance of missing this bad boy!

Controller Architecture and Documentation

Details of the hardware architecture for the Solo Brew Controller can be found here. This includes circuit diagrams and PCB designs for the controller, a description of the microcontroller architecture, and a parts list for building the project. For this project, I use the Arduino Mega 2560, Rev 3, microcontroller. This is coupled with an LT7683 chip to drive the graphics displayed on a 7in 1024x600 pixel TFT Touch Display.

The overall software architecture can be found here. A collection of UML diagrams describe structure of the C++ software that runs on the Mega 2560 microcontroller. More detailed information about the software functions and classes within the software baseline can be found in the Software Design Documentation.

Building this Controller requires an understanding of:

  • Construction of Printed Circuit Boards (PCBs)
  • Understanding of Mega 2560 microcontrollers
  • Knowledge of C/C++
