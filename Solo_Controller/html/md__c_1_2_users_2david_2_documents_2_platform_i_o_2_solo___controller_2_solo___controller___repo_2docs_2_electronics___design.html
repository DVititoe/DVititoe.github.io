Electronics Design

The electrical components of the Solo BIAB Controller are split between two enclosures: Display and Sensor Breakout Enclosure. The 1024x600 pixel touch screeen and associated control electronics are all contained within the Display Enclosure. There are four primary components to the control electronics that interface as shown in the following figure.

This section focuses on the design of the control electronics. Circuit diagrams showing the full wiring of the Display enclosure can be found here.

Electronics Components

The four components that make up the control electronics are as follows:

Mega 2560, Rev 3 Microcontroller

The Arduino Mega 2560, Rev 3 microcontroller is the "brains" for the control electroncs. The software loaded onto the microcontroller is here. Documentation describing the sofware architecture is here.

7 inch 1024x600 TFT Touch Display Module

The Solo Controller uses a 7", 1024x600 pixel TFT display from <a href="https://www.buydisplay.com/" >BuyDisplay.com</a>. Model <a href="https://dvititoe.github.io/Solo_Controller/html/docs/ER-TFT070A2-6-5633_Datasheet.pdf" >ER-TFT070A2-6-5633</a> was selected. This display has built in Capacitance Touch Panel (CTP) capabability. Information about the CTP capability can be found in the <a href="https://dvititoe.github.io/Solo_Controller/html/docs/FT5316.pdf" >CTP Datasheet</a> and the <a href="https://dvititoe.github.io/Solo_Controller/html/docs/FT5x16_registers.pdf" >CTP Data Register</a> documentation. <center><img src="../images/TFT_Display.png" alt="TFT Display" width="90%"></center> The display has a built in MicroSD Card Interface, which is not used by the Solo Controller. It can also be upgraded, by soldering a 128M bit (SOP8) ROM chip to the board. This gives more than enoungh non-volitile storage capacity to hold the imagery used by the Solo Controller software. Finally, the display includes a <a href="https://dvititoe.github.io/Solo_Controller/html/docs/LT7683.pdf" >LT7683</a> graphics chip. the LT7683 has 128Mb of built in RAM.
