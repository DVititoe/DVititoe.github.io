Electronics Design

The control electronics for the Solo Controller are contained within the Display box shown here. The control electronics manage the collect temperature and volume sensor information, manage output to heating elements, pumps, and auxillary devices, handle user inputs via the touch screen interface, and manage kettle temperatures via PIDs.
